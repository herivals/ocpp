Réseau

GET

/health

Vérifie l'état du serveur et retourne les bornes actuellement connectées.

POST

/start

Démarre une transaction à distance sur une borne (RemoteStartTransaction).

POST

/stop

Arrête une transaction en cours sur une borne (RemoteStopTransaction).

POST

/reset

Redémarre une borne. Soft = attend la fin de session, Hard = reboot immédiat.

GET

/chargers/:id/config

Récupère toutes les clés de configuration OCPP d'une borne (GetConfiguration).

GET

/chargers/:id/config/:key

Récupère la valeur d'une clé de configuration spécifique.

POST

/chargers/:id/apply-config

Applique en masse un fichier JSON de configuration à une borne (ChangeConfiguration en boucle).

POST

/chargers/:id/configure-wifi

Configure le WiFi d'une borne via OCPP ChangeConfiguration (clé WifiLinkInfo).

BATCH

Configuration en chaîne

Sélectionnez plusieurs bornes connectées et appliquez une opération à toutes simultanément.
